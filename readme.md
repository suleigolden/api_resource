# Student API

> A robust API using Laravel 5.8 and API resources. 

## Stallation

``` bash
# Install Dependencies
composer install

# Run Migrations
php artisan migrate

```

## API Endpoints

### Store student record
``` bash
POST api/store/student
```
### Get single student
``` bash
GET api/students/{id}
```
### Get all student
``` bash
GET api/students/
```

### Delete article
``` bash
DELETE api/article/{id}
```
